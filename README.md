# Objective:
- Live update of USD into INR from yahoo finance.
- Help students from India send money to the USA with maximum profit and no loss.

# Environment: MS Excel
- Run a background query to fetch live data in a spreadsheet form from Yahoo Finance.
- Performed what-if analysis to compare data for the past 6 months.
- Data visualization using live line charts and future profits and losses.

# Conclusion:
This Excel model helps Indian parents/students to transfer money any day by evaluating the same day Dollar value with past six months values and also future Dollar INR fluctuating rates.
